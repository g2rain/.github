## Hi there 👋
我们是一个开源的SaaS平台——**开源谷雨SaaS平台**（英文名称：grain rain，简称 g2rain）。本平台以**多租户**、**应用交付**、**商务计费**为核心能力，提供完整的系统架构解决方案。

平台采用微服务架构，通过**网关**和**推送中心**实现同步和异步两种数据交付方式，并提供了一致授权与鉴权，基于 **DPoP**、**IAM**、**Lua** 等技术实现安全与能力扩展。

项目遵循 **DDD（领域驱动设计）** 理念，为便捷开发业务功能提供了：
- 接口交互规范（common 包）
- Java 项目生成器
- 代码生成器
- 前端应用页面生成器
使用的技术包括：

## 后端：
Java(JDK21), Spring(Srping boot4.0), Spring cloud, Spring gateway, Mysql, Redis等常用组件
## 前端：
Typescript, Vue3, qiankun(微前端), Vite5, Element plus等组件
